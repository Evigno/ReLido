# Sistema de Gestão de Obras - Projeto ReLido

Este projeto consiste em um programa em Python que implementa um CRUD (Create, Read, Update, Delete) para o gerenciamento da entidade Obra no contexto do sistema ReLido. O objetivo é permitir que administradores cadastrem e gerenciem livros no acervo.

## Funcionalidades

O sistema permite:
- Inserir uma nova obra (com Nome, Gênero, Edição e ID do Administrador).
- Consultar todas as obras cadastradas.
- Atualizar dados de uma obra existente.
- Excluir uma obra do banco de dados.

- Linguagem: Python 3
- Banco de Dados: SQLite3
